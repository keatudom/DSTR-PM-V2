**หมายเหตุ:**
- `─` `═` (box-drawing 2,278 ครั้ง) = เส้นคั่นตกแต่ง → ใช้ `border`/`hr` ของ design system แทน ไม่ใช่ไอคอน
- จุดสีสถานะ (🔴🟡🟢) → ทำเป็น dot ด้วย `.badge` สีหรือ `<span>` กลมเล็ก ไม่ใช่ Lucide (Lucide เป็นเส้น outline ไม่มีสีทึบ)
- ไอคอนหมุน (loading): `<i data-lucide="loader-2" class="svg-spin"></i>` — class `svg-spin` มีใน design-system.css
- ตรวจชื่อไอคอนได้ที่ https://lucide.dev/icons (เราล็อก **lucide v1.24.0** self-host ที่ `vendor/lucide/lucide.min.js`)
