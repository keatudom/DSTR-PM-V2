# Staff Photos — about.html

วางรูป profile ของพนักงาน 7 คนในโฟลเดอร์นี้ ระบบจะดึงไปแสดงใน `about.html` อัตโนมัติ

## ขนาดและไฟล์ที่แนะนำ

- ขนาด: **อย่างน้อย 400×400 พิกเซล** (จะ render ที่ 88px / 60px แต่ retina จะคมขึ้น)
- ฟอร์แมต: `.jpg` (preferred — ขนาดเล็กกว่า PNG สำหรับรูปคน)
- น้ำหนัก: ไม่เกิน 200KB ต่อรูป

## วิธี extract รูปจาก PDF (กรณียังไม่มีไฟล์แยก)

1. เปิด `Desktop/About us DSTR/3.Executive committee organization chart.pdf`
2. ไปที่หน้า 4 (Executive chart — รูปวงกลม) หรือหน้า 6-7 (Collaborative Working Team — รูปสี่เหลี่ยม)
3. ใช้ **Snipping Tool** (Windows) — กด `Win+Shift+S` แล้วเลือกเฉพาะรูปคน
4. Save เป็นชื่อตามตารางด้านบน
